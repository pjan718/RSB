NAVT Language Directory

This directory contains various localization files used by NAVT. 
The file in use depends on the WPLANG setting in your wp-config.php file.

Set WPLANG to one of the following:


WPLANG                Language          Country            Translator
----------------------------------------------------------------------
'en' or 'en_US' ..... English ......... United States [default language]
'sk' or 'sk_SK' ..... Slovak  ......... Slovakia ......... Tomas Forro
'de' or 'de_DE' ..... German  ......... Germany .......... Rico Neitzel
'ru' or 'ru_RU' ..... Russian ......... Russia ........... Dmitriy Kostromin


If you think you've found an error, please contact me at greg @ gbellucci . us
